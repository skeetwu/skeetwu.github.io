心静，才能做出好事情

By 武鹏 • On 2017年7月17日 • In 技术总结分享

最近媳妇正在忙于找工作，心情一直不大好，周末在家，把孩子交给妈妈看着，打算带她去山上的小庙散散心。

小庙在半山腰，属于前照后靠，左环右抱的好风水，之前经常去，驱车走到半路，发现在修路，本来的水泥路也被完全挖断了，问了旁边施工的大哥，大哥说只能从南边的路上去，无奈折返回来，绕了两个山包才到了山脚下。道路不畅，显得四处都很萧条，原本干净的石阶上，竟也冒出了青草，刚下过雨，感觉草是青绿色调的。

沿阶而上，媳妇和我都许久未运动，原来轻松的几步路，竟然需要停两次才能走完。到了庙们，凉快，风很凉而且吹的柔和又持续，总是能带给我一种闹市中久违的宁静。

庙里在门廊里做神像，估计是利用道路不通，香火略少的时间搞一搞基础建设。一个年轻的小哥正站在马凳上认真的给神像做手里的环形武器，嘴里抽着烟，手机放着地上，正放着评书，裤子上满是泥土，但是感觉到的不是邋遢，而是那种工匠的执着。我叫了他几声，他才意识到我们进来了，笑了笑打了个招呼，我们就进去了。

上了上香，许了几个愿望，我就出来了，媳妇还在继续，她的愿望更容易实现，所以得多拜一下，我转了一圈又回到门廊，小哥已经从马凳上下来，斜躺在椅子上，望着远处，手里夹着烟，小凳子上放着烟盒和打火机，手机仍然放着刚才那段评书。看到我过来了，稍微坐正了一下，我看媳妇出来还早，就给他多聊了几句。

小伙是河南人，十四从师学得这门手艺，同样也是面临无法传承下去，高兴的事情这两样当地政府已经将这们手艺申请非遗，每年还会举办一些展会和比赛，有一些抬头之势。做这个行当非常辛苦，常年不着家，这一尊神像，前后需要三个月到半年的时间才能全部完工，工艺流程相当繁复。但是小哥的言语中，充满了对这项工作的热爱，充满的对泥塑的热忱，看样也是会经常去参加一些展会。抱怨也是有的，孤独，重复，不断的等待，也是无法回避的工作状态。好在看似他已经找到了排解方法，并已经习惯。

小伙说了许多泥塑的内容，上色后并不是最美的，最质朴的才是最美的，我运气好，刚好看到了泥塑的本身美。确实与漆后的华彩不同，少了一份喧嚣，多了份朴实。

此时，他的伙伴叫他吃午饭，他讷讷的在一个水桶里洗了洗手，没有毛巾直接在身上擦了擦，然后笑着跟我告别，也许是好久没和陌生人沟通过，有些不舍，又简单聊了几句，伙伴高声催促，他不得不笑嘻嘻的走了。

这也是种修行，一种身心共鸣的修行，耐得住寂寞，控得住心境，才能做出好的作品。

美好的一天，晚上戳子肉，走起！

Spring reference上关于IoC的一段解释，好绕。

By 武鹏 • On 2017年7月4日 • In 技术总结分享

Inversion of Control (IoC) [1] principle. IoC is also known as dependency injection (DI). It is a process whereby objects define their dependencies, that is, the other objects they work with, only through constructor arguments, arguments to a factory method, or properties that are set on the object instance after it is constructed or returned from a factory method. The container then injects those dependencies when it creates the bean. This process is fundamentally the inverse, hence the name Inversion of Control (IoC), of the bean itself controlling the instantiation or location of its dependencies by using direct construction of classes, or a mechanism such as the Service Locator pattern.

自动下载七牛日志CDN日志

By 武鹏 • On 2017年6月22日 • In 技术总结分享

最近在做日志分析，发现有些静态页面的内容是携带重要参数的，七牛的cdn提供了日志下载接口，但是没有完全自动化。

跟客服沟通后，确实是没有，没办法只能自己造个轮子，决定把七牛的log拿过来，自己处理分析。

关于七牛日志的一些说明和描述在这里

https://developer.qiniu.com/fusion/api/1226/download-the-log

所以最好是写个脚本，每天中午十一点自动执行。单纯的shell脚本无法全部完成，所以还得结合一些python。不过关于日志下载列表这块，七牛的sdk有个坑，待会说。

#!/bin/sh
 
#retrive yesterday logs from qiniu cdn every 11:00 am
#see https://developer.qiniu.com/fusion/api/1226/download-the-log
 
yesterday=`date -d -1day +%Y-%m-%d`  
 
if [ ! -z "$1" ] ;then
  yesterday="$1"
else
  echo "no input, user yesterday!"
fi
 
 
echo $yesterday
echo "--------------  $yesterday 's logs begin to collect"
cd /opt/qiniu_logs/logs/
rm -rf *.log
 
python /opt/qiniu_logs/downloadlogs.py $yesterday
echo "-------------- create download shell "
 
while read line
do
$line
done < '/opt/qiniu_logs/target.list'
gzip -d *.gz
for i in `ls`; do mv -f $i `echo $i".log"`; done
echo "-------------- download and unzip "
 
java -jar /opt/qiniu_logs/a.jar demo.LogCollecter

python脚本主要的工作是把七牛日志的url列表获取到。

url格式：

http://fusionlog.qiniu.com/v2/app.iuu.travel_2017-06-21-08_part-00000.gz?e=1529561090\&token=557TpseUM8ovpfUhaw8gfa2DQ0104ZScM-BTIcBx:JJyYu9oG6r44TUalBz_-1mBTuzM

拼接成一个wget的语句写入到文本文件中，但是这个地方一定要注意生成的时候 wget的url要转移?，这个坑搞了好久才跳出来，调试时明明在本地可以，到服务器上就不行了，后来才发现服务器的报错提示无授权，才发现token传的不对。

wget -q http://fusionlog.qiniu.com/v2/app.iuu.travel_2017-06-21-08_part-00000.gz?e=1529561090\&token=557TpseUM8ovpfUhaw8gfa2DQ0104ZScM-BTIcBx:JJyYu9oG6r44TUal
Bz_-1mBTuzM -O app_2017-06-21_9.gz
